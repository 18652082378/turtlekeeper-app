# 壳友手账 App 原型

打开 `index.html` 即可预览。

已完成：

- 首页统计：总数量、在养、健康、生病、品种数、最近新增。
- 首页模块入口：新增乌龟、乌龟列表、品种管理、提醒事项、养护日历、繁殖管理。
- 品种选择：中文名称、图片、按首字母分组、搜索、添加到饲养品种。
- 饲养品种管理：展示已添加品种，支持删除；删除不影响已有乌龟数据。
- 乌龟列表：新增乌龟、品种筛选、默认排序、最新添加、按体重排序、删除乌龟。
- 新增乌龟：照片上传、品种选择、性别、重量、背甲长度、状态、健康状态、入手日期、来源、价格、备注。
- 提醒事项：新增、编辑、删除；每次新增或编辑都会自动更新操作时间。
- 我的：基础个人中心页面。

说明：

- 当前是可运行的移动端网页原型，数据保存在浏览器本地。
- 品种图片使用在线图片链接，正式上架前建议替换为已授权素材或自有素材。
- 后续要上架 Apple Store，需要继续封装为 iOS 工程，并补齐隐私政策、图标、启动图、账号体系和审核材料。
