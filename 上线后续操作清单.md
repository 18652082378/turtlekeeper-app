# 壳友手账上线后续操作清单

## 1. 安全处理

- 如果任何 AccessKey Secret 曾经出现在聊天、截图、GitHub 或 `.env.example` 中，立即在阿里云 RAM 中禁用并删除，重新创建新的 AccessKey。
- 生产密钥只写入服务器 `server/.env`，不要提交到 GitHub。

## 2. 域名和解析

在阿里云 DNS 添加：

```text
主机记录：api
记录类型：A
记录值：你的 ECS 公网 IP
```

最终接口域名：

```text
https://api.turtleworld.cn
```

App Store / 备案后台服务域名填写：

```text
api.turtleworld.cn
```

## 3. 云服务器部署

以下以 Ubuntu 22.04/24.04 为例。

```bash
sudo apt update
sudo apt install -y nginx git curl
curl -fsSL https://deb.nodesource.com/setup_24.x | sudo -E bash -
sudo apt install -y nodejs
sudo npm install -g pm2
```

拉取项目：

```bash
cd /www
sudo git clone https://github.com/18652082378/turtlekeeper-app.git
sudo chown -R $USER:$USER turtlekeeper-app
cd turtlekeeper-app
```

创建生产环境变量：

```bash
cp server/.env.example server/.env
nano server/.env
```

示例：

```env
SMS_PROVIDER=aliyun-pnvs
SMS_MOCK=false
ALIYUN_ACCESS_KEY_ID=新的AccessKeyId
ALIYUN_ACCESS_KEY_SECRET=新的AccessKeySecret
ALIYUN_SMS_SIGN_NAME=短信签名名称
ALIYUN_SMS_TEMPLATE_CODE=100001
ALIYUN_SMS_TEMPLATE_PARAM_KEY=code
ALIYUN_SMS_TEMPLATE_MINUTES=5
ALIYUN_SMS_CODE_LENGTH=6
ALIYUN_SMS_VALID_TIME=300
ALIYUN_SMS_COUNTRY_CODE=86
HOST=127.0.0.1
PORT=8787
```

启动后端：

```bash
pm2 start ecosystem.config.cjs --env production
pm2 save
pm2 startup
```

## 4. Nginx 和 HTTPS

复制 Nginx 配置：

```bash
sudo cp deploy/nginx-api.turtleworld.cn.conf /etc/nginx/sites-available/api.turtleworld.cn
sudo ln -s /etc/nginx/sites-available/api.turtleworld.cn /etc/nginx/sites-enabled/api.turtleworld.cn
sudo nginx -t
sudo systemctl reload nginx
```

申请 HTTPS 证书：

```bash
sudo apt install -y certbot python3-certbot-nginx
sudo certbot --nginx -d api.turtleworld.cn
```

## 5. 接口测试

```bash
curl -X POST https://api.turtleworld.cn/api/sms/send \
  -H "Content-Type: application/json" \
  -d '{"phone":"你的手机号","purpose":"register"}'
```

成功时应返回：

```json
{"ok":true,"mode":"aliyun","expiresIn":300}
```

## 6. GitHub 推送

```bash
git add .
git commit -m "Prepare production deployment"
git push
```

## 7. Codemagic 打包

- 连接 GitHub 仓库。
- 添加 App Store Connect 集成，名称必须是 `app_store_connect`。
- 上传 Apple Distribution 证书和 App Store provisioning profile。
- 运行 `iOS TestFlight` 工作流。
- 成功后会上传到 TestFlight。

## 8. App Store Connect

建议填写：

```text
隐私政策 URL：https://api.turtleworld.cn/privacy.html
支持 URL：https://api.turtleworld.cn/support.html
后台服务域名：api.turtleworld.cn
Bundle ID：com.turtlekeeper.app
```

审核备注里提供测试账号：

```text
手机号：审核专用手机号
验证码：请联系开发者，或配置固定测试验证码
```
